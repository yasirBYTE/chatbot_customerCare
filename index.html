Architecture

From query to response

Four steps transform a customer message into an intelligent answer.

01

Input

The customer types a message in the Streamlit interface. It is captured and forwarded to the bot engine for processing.

02

Preprocessing

Text is lowercased, stripped of punctuation, tokenized, and filtered against NLTK's English stopword list — retaining only meaningful terms.

03

Classification

A TF-IDF vectorizer converts the cleaned text into numerical features. Logistic Regression predicts the intent with a confidence score.

04

Response

Confidence ≥ 0.4 returns a curated response. Below that, a graceful fallback directs the user to human support channels.
