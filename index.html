How It Works

The architecture behind the bot — from user input to intelligent response.

1

User sends a message

The user types a question in the Streamlit chat interface. The message is captured and passed to the bot engine.

2

Text preprocessing

The raw text is lowercased, stripped of punctuation and special characters, tokenized into words, and filtered against NLTK's English stopword list.

3

Intent prediction

The cleaned text is vectorized using a pre-trained TF-IDF model (unigrams + bigrams, max 5000 features). A Logistic Regression classifier predicts the intent with a confidence score.

4

Response generation

If confidence is ≥ 0.4, a random response from the matched intent is returned. If confidence < 0.4, a polite fallback is shown suggesting contact support.

Run Locally

Get the chatbot running on your machine in under a minute.

1

Clone the repo

git clone https://github.com/yasirBYTE/chatbot_customerCare.git
cd chatbot_customerCare
2

Install dependencies

pip install -r requirements.txt
3

Run the app

streamlit run app.py

Open http://localhost:8501 in your browser. The model auto-trains on first launch.

4

Customize intents (optional)

Edit intents.json to add or modify training examples and responses. Delete model.pkl and restart to retrain.
